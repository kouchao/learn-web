现在开始我们从 0 开始学习 HTML 的相关知识，你所看到的页面就是用 HTML 来写的，它看起来很简单，没有布局，也没有样式，正如 0 基础的我们，它会随着我学习的知识逐步完善。 介绍一下 HTML ，它是超文本标记语言 (Hyper Text Markup Language)，用来描述页面的，一般会用 index.html 来作为首页，新建一个 index.html 文件，那么就可以开始了。 不同的内容在不同的标签内，标签是由尖括号包裹的。 如 <html></html>，像是这种我们叫做闭合标签。 不同标签的语义不同，另外他会有浏览器给到的默认样式和功能。看起来我们的内容都在一起，很难区分开段落，让我们首先学习一个段落标签，<p>段落元素

<p>元素表示文本的一个段落。它可以另起一行，并且与上面下面隔开一段距离，我们可以更好的来阅读。

恭喜你已经会了第一个 HTML 标签。你可以在 index.html 中写一下试试，并使用浏览器打开这个文件。

通过使用标签，让内容有了结构，也叫内容结构化，它的好处显而易见，阅读清晰了很多。

HTML 也有自己的结构，我们看一下 HTML 的结构，如下：

<!doctype html> <html> <head> <meta charset="UTF-8"> </head> <body> ... </body> </html>

我在原文件中有写换行，但是在这里他不会显示，因为在 HTML 中的空格、制表符、换行都被视为“空白字符”

我们想让文字按照原文件中的编排，可以使用<pre>元素，表示预定义格式文本，文本中的“空白字符”都会显示原本的样子

    <!doctype html>
    <html>
    <head>
    <meta charset="UTF-8">
    </head>
    <body>
    ...
    </body>
    </html>

上面就是放在<pre>标签中的效果，如果你正在跟着书写，那么你得到的结果应该不是这样的。

思考一个问题，你在源文件中写 <pre> 的时候，浏览器怎么知道应该显示 <pre> 还是它的结构？

按照约定，默认显示的就是它的结构，而我们想显示尖括号字符，需要将字符转移，再与 HTML 约定的字符发生冲突的时候使用，比如我们刚才的尖括号应该写成 &lt;pre&gt;

上文中需要转移的有“<” “>” “"”，现在你应该就可以显示出来了

让我们回到 HTML 的结构中

<!doctype html> 中，<!doctype>是一种指令，它告诉浏览器页面使用哪个 HTML 版本的规范来渲染文档。当前使用的文档类型是 HTML5

<html>元素表示一个 HTML 文档的根（顶级元素），所以它也被称为根元素。所有其他元素必须是此元素的后代。

<head>元素 规定文档相关的配置信息（元数据），包括文档的标题，引用的文档样式和脚本等。

<meta>元素表示那些不能由其他 HTML 元相关（meta-related）元素表示的元数据信息。比如我们就使用 charset 声明了文档的字符编码为 UTF-8

<body>元素表示文档的内容。

在现代的编辑器中，如 webstorm，vscode，通常只需要输入一个!,然后按 tab 键，就会生成一段默认的 HTML 结构

现在，我们已经知道了一个 HTML 是什么样子的，学会了使用<p>来表示一个段落，让我们继续探索其他的标签。
